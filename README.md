# 🛍️ Eclypse — Modern E-Commerce Web App

A full-stack, responsive e-commerce web application built with **React**, **TailwindCSS**, **TypeScript**, and **Node.js/Express**.  
Designed to showcase clean code, modern UI/UX, and scalable architecture.


---

## ✨ Features

- ⚡ **Responsive UI**: Works beautifully on desktop, tablet, and mobile.
- 🎨 **Modern Design**: Monochrome theme with red accents, glassy cards, and subtle animations.
- 🛒 **Product Gallery**: Paginated, 15 products per page, with image hover effects.
- 🧾 **Cart & Checkout**: Add to cart, view cart, and checkout pages.
- 📝 **Waitlist & About**: Join waitlist, learn about the brand.
- 💬 **Testimonials**: Rotating customer reviews with avatars.
- 🔗 **API Backend**: Node.js/Express backend serving dummy data via REST API.
- 🌐 **Deployed on Railway/Vercel**: Live demo with separate frontend and backend.
